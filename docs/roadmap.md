# 医美 AI Agent 项目实施路线图

> Version: 0.1
> Date: 2026-05-08
> 配套设计文档：[design.md](./design.md)

---

## 总览

项目拆分为 **11 个阶段（Phase 0-10）**，按依赖顺序推进。每个阶段都有明确的产出与里程碑。

### 推荐起步路径
- **最小可演示（2-3 周）**：Phase 0 → 1 → 2 → 7（简化）→ 4
- **完整 MVP（4-6 周）**：Phase 0 → 1 → 2 → 3 → 4 → 5 → 7
- **生产上线（2-3 月）**：完整 Phase 0-9
- **专业化（可选）**：Phase 10

---

## 🏗️ 基础建设

### Phase 0：环境与基础设施搭建（1-2 天）

**目标**：本地开发环境一键启动

**任务**：
- [ ] Git 仓库初始化、`.gitignore`、`README.md`
- [ ] Python 虚拟环境（推荐 `uv` 或 `poetry`）
- [ ] `.env` 与 `.env.example` 配置管理
- [ ] Docker Compose 编排
  - PostgreSQL + pgvector 扩展
  - Redis
  - Langfuse（self-host）
- [ ] 申请 / 配置外部服务
  - DeepSeek API key
  - bge embedding（SiliconFlow API 或本地）
- [ ] CI 骨架（GitHub Actions / 本地 pre-commit hook）

**产出**：`docker compose up` 一键启动所有依赖服务

---

### Phase 1：后端骨架 + LLM 接入（2-3 天）

**目标**：能跑通流式对话

**任务**：
- [ ] FastAPI 项目结构（routers / services / models / core 分层）
- [ ] Pydantic 配置类（读取 `.env`）
- [ ] DeepSeek V4 客户端封装（基于 OpenAI SDK 兼容协议）
- [ ] 健康检查接口 `/health`
- [ ] SSE 流式 `/api/v1/chat` 接口（先纯 LLM，无 RAG）
- [ ] 基础日志系统

**产出**：`curl` 能调通流式对话接口

---

## 🧠 核心能力（MVP 关键）

### Phase 2：知识库与 RAG 基础（3-5 天）

**目标**：基于知识库回答带引用的问题

**任务**：
- [ ] PostgreSQL 表结构创建（`documents`、`doc_chunks`）
- [ ] Alembic 数据库迁移
- [ ] 文档解析（markitdown → Markdown）
- [ ] 文本切分（chunk_size=512, overlap=64）
- [ ] bge-large-zh embedding 接入
- [ ] pgvector 入库脚本（含 HNSW 索引）
- [ ] LlamaIndex 检索器封装（向量召回 Top K）
- [ ] 灌入种子资料（50-100 篇热门项目：玻尿酸、肉毒、热玛吉、超声炮等）
- [ ] `/chat` 接口集成 RAG，回答附 Citation

**产出**：能回答"玻尿酸维持多久？"这类问题，带准确引用

---

### Phase 3：Agent 编排（LangGraph）（5-7 天）

**目标**：完整多轮咨询流程

**任务**：
- [ ] 定义 `ConsultState`（TypedDict）
- [ ] 实现 6 个节点
  - `Intake`：意图识别 + 用户信息抽取
  - `SafetyGate`：敏感问题拦截
  - `Retrieve`：LlamaIndex 检索
  - `RiskAssessment`：禁忌症评估
  - `Recommend`：方案生成
  - `Disclaim`：免责声明
- [ ] 节点间条件路由（信息不全 → 反问）
- [ ] 多轮对话状态持久化（PG `sessions`、`messages` 表）
- [ ] 工具注册（search_knowledge / lookup_project / check_contraindication）
- [ ] LangGraph checkpoint 集成

**产出**：能完成"收集信息 → 检索 → 推荐方案"完整流程

---

### Phase 4：安全合规层（2-3 天）

**目标**：上线前的安全底线

**任务**：
- [ ] 输入侧
  - 敏感词过滤库（色情、政治、辱骂）
  - 诊断意图分类器（"我得了 XX 病吗"→ 拒答）
  - PII 脱敏（手机号、身份证号入库前打码）
- [ ] 输出侧
  - 禁用词检测（"诊断""处方""治愈""保证效果"）
  - 强制免责声明追加
  - Citation 强制（无引用则拒答）
- [ ] `audit_logs` 表 + 审计落库
- [ ] 拒答兜底回复模板

**产出**：违规内容拦截率 ≥ 95%，所有敏感事件可追溯

---

## 📊 工程化

### Phase 5：可观测性（2 天）

**目标**：每次对话的完整链路、成本、延迟可见

**任务**：
- [ ] Langfuse SDK 接入
  - 每个 LLM 调用埋点
  - Prompt 版本管理
  - Token 消耗与成本统计
- [ ] Agent 节点级 trace
- [ ] Prometheus 指标暴露（QPS、错误率、延迟分布）
- [ ] Grafana 看板（API 健康、LLM 调用、数据库）
- [ ] 业务指标统计
  - 拒答率
  - 引用命中率
  - 用户满意度（点赞率）

**产出**：能在 Langfuse 看到每次对话的完整调用链与成本

---

### Phase 6：检索质量优化（3-5 天）

**目标**：检索准确率提升 20-30%

**任务**：
- [ ] bge-reranker-v2-m3 接入（精排）
- [ ] BM25 关键词检索（PostgreSQL FTS 或 Elasticsearch）
- [ ] 混合检索（RRF 融合向量 + BM25）
- [ ] 元数据过滤（按项目类别、文档权威等级）
- [ ] 文档权威分级标注（L1-L4）
- [ ] 检索评测集构建（100+ 标准问答对）
- [ ] A/B 评估框架

**产出**：评测集上 Recall@5 ≥ 90%

---

## 🎨 用户侧

### Phase 7：前端 MVP（5-7 天）

**目标**：可演示的产品形态

**任务**：
- [ ] Next.js 14 + TypeScript 项目初始化
- [ ] Tailwind CSS + shadcn/ui 集成
- [ ] Vercel AI SDK 接入（流式聊天）
- [ ] 聊天界面
  - 消息列表（流式渲染）
  - Citation 卡片展示
  - Markdown 渲染
- [ ] 会话历史侧边栏
- [ ] 移动端适配
- [ ] 用户反馈（点赞 / 点踩）
- [ ] 隐私协议、免责声明弹窗

**产出**：可对外演示的 Web 产品

---

## 🚀 上线运营

### Phase 8：知识库扩充与运营工具（持续）

**目标**：知识库质量持续提升

**任务**：
- [ ] 系统化收集医美资料（目标 1000+ 篇）
  - 公开论文（PubMed、知网）
  - 临床指南（中华医学会、ISAPS）
  - 药械说明书（NMPA/FDA）
  - 厂商资料、教科书
- [ ] 专家审核后台（管理界面，标注权威等级、分类）
- [ ] 文档版本管理（更新追溯）
- [ ] 知识库更新流水线（增量 embedding）
- [ ] Bad case 收集机制（用户点踩自动入库）
- [ ] 回归测试集（每次模型/prompt 变更跑一遍）

**产出**：覆盖主流医美项目 80%+，含审核流程

---

### Phase 9：部署与上线（3-5 天）

**目标**：公网可访问

**任务**：
- [ ] Docker 镜像构建
- [ ] 云服务器选型（阿里云 / 腾讯云 / 火山引擎）
- [ ] 域名注册、HTTPS（Let's Encrypt）
- [ ] Nginx 反向代理
- [ ] 数据库备份策略（每日全量 + binlog）
- [ ] 限流（Redis 令牌桶）
- [ ] WAF（云厂商提供）
- [ ] 隐私协议、用户协议（法务审核）
- [ ] 生成式 AI 服务备案准备（如需 ToC 商业化）

**产出**：公网域名访问、HTTPS、稳定运行

---

## 🔬 专业化（可选）

### Phase 10：专业化与本地化（月级）

**触发条件**：客户要求数据不出网 / 用户量大需降本 / 需要深度领域优化

**任务**：
- [ ] 自部署 Qwen2.5-14B（vLLM）
- [ ] GPU 资源采购 / 云租用
- [ ] 医美专家审核语料 → LoRA 微调
- [ ] 真实医生人工兜底（复杂问题转人工）
- [ ] 用户画像持久化与长期记忆
- [ ] 多模态评估（用户上传照片 → 项目推荐，注意医疗影像合规）

**产出**：达到准专业咨询水平，数据不出网
